Columns (Attributes):

gender:

Type: Categorical (Nominal)
Description: Gender of the student (e.g., 'male', 'female').

race/ethnicity:

Type: Categorical (Nominal)
Description: The race/ethnicity group of the student (e.g., 'group A', 'group B', etc.).

parental level of education:

Type: Categorical (Ordinal)
Description: The highest level of education attained by the student’s parents (e.g., 'associate’s degree', 'bachelor’s degree', etc.).

lunch:

Type: Categorical (Nominal)
Description: Type of lunch the student receives (e.g., 'standard', 'free/reduced').
test preparation course:

Type: Categorical (Nominal)
Description: Whether the student completed a test preparation course (e.g., 'none', 'completed').
math score:

Type: Integer (Discrete)
Description: Score achieved by the student in math.
reading score:

Type: Integer (Discrete)
Description: Score achieved by the student in reading.
writing score:

Type: Integer (Discrete)
Description: Score achieved by the student in writing.


Row Count:
Total Rows: Based on the dataset size, the total number of rows corresponds to the number of students included.

General Notes:
The dataset contains both categorical and continuous data.
The focus of the dataset is on student performance, specifically their scores in math, reading, and writing, alongside demographic details.
